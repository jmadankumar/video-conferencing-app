# Video Conferening App

![Video conferencing app landing page](./docs/images/landing.png)

## Getting Started

### Prerequisite

- NodeJS (>=12+)
- Yarn (>=1.19+)

### Technologies used

- Webrtc
- ReactJS
- Typescript
- NodeJS
- Websocket
- Tailwindcss
- styled-components

### Run client

```
cd client
yarn start
```

### Run server

```
cd server
yarn start
```
